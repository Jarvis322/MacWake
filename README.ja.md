# 🔋 MacWake

[English](README.md) | [简体中文](README.zh-Hans.md) | [Türkçe](README.tr.md) | [日本語](README.ja.md) | [한국어](README.ko.md)

**MacWake** は、バッテリーの詳細な健全性、使用状況の分析、充電習慣を追跡するために macOS 向けに設計された、洗練されたメニューバーおよびデスクトップウィジェットアプリケーションです。Swift と SwiftUI で構築され、現代的な macOS デザインガイドライン（グラスモーフィズム、鮮やかなエフェクト）を忠実に採用しています。

## 🖥️ Dynamic Island とメニューバー

ノッチ内には **Dynamic Island** があり、折りたたみ時はハードウェアになじみます。ホバーまたはクリックすると弾むようなスプリング動作で展開し、触覚フィードバックを返します。リアルタイムの電力、バッテリーの健全性、バッテリー温度を表示するほか、ウィジェット、セッションのリセット、アニメーション、通知のクイック切り替えも利用できます。

## ✨ 機能

*   **🔋 充電上限（Apple Silicon M シリーズ全機種）：**
    *   充電上限を 50%～95% の任意の値に設定し、長期的なバッテリーの劣化を抑えます。
    *   チップごとに最適な方式を選択します。M1/M2/M3 ではクリーンな充電抑制（CHTE/CH0C）、M4 ではアダプター制御（CHIE）を、小さな公証済みバックグラウンドヘルパーを介して使用します（初回のみ承認が必要で、パスワード入力は求められません）。
    *   **⛵ セーリングモード：** 上限付近で細かな充電を繰り返さず、下限までバッテリー残量を下げてから再充電します。サイクル数と発熱を抑えられます。
    *   **🧪 ディープバッテリーキャリブレーション：** 約 15% まで放電し、100% まで充電してから 1 時間保持するフルサイクルで、残量ゲージを再調整します。スケジュールまたは「今すぐキャリブレーション」から実行でき、現在のフェーズとキャンセルボタンも表示されます。
*   **⌨️ コマンドラインツール：**
    *   設定から `macwake` CLI をインストールし、Terminal で充電を制御できます：`macwake status`、`charging on|off`、`adapter on|off`、`energy auto|low|high`、`fan auto|<rpm>`。
*   **⚡️ エネルギーモード：**
    *   macOS のエネルギーモード（自動 / 低電力 / 高性能）をメニューから直接切り替えられます。高性能は、対応する Mac でのみ表示されます。
*   **🌀 モニタータブ — ファン + 上位アプリ：**
    *   ファンの状態、1 時間の回転速度履歴、手動の目標 RPM（実験的機能。92°C を超えると自動的にシステム制御へ戻ります）を専用タブに表示します。
    *   実行中の上位アプリを CPU または RAM で順位付けし、Activity Monitor / iStat Menus のように必要なときだけサンプリングします。
*   **🎯 スマート充電の追加機能：**
    *   ワンタップの上限プリセット（60/70/80/90%）、旅行日に便利な 1 回限りの「今回だけ 100% まで充電」、スケジュールによるフル充電を利用できます。毎日、指定した時刻に 100% になるよう準備します。
    *   温度ガードは 40°C を超えると充電を一時停止し、バッテリーが冷えると再開します。
    *   macOS 標準の通知より前に、指定した残量でカスタム低バッテリー通知を表示します。
*   **⚡️ Shortcuts とウィジェット：**
    *   Shortcuts アプリのアクション：充電上限を設定、今回だけ 100% まで充電、クリーニングモードを開始、バッテリー状態を取得。
    *   バッテリーリング、健全性、温度、上限の状態を表示するネイティブ macOS ウィジェット（小・中）。
    *   健全性/サイクル履歴、セッション、アダプターログを CSV として書き出せます。
*   **🎛️ カスタマイズ可能なメニューバー：**
    *   メニューバー項目に表示する内容（アイコン、バッテリー残量（%）、電力/時間、推定残り時間、温度）を、ライブプレビューで正確に選択できます。
*   **🏝️ Dynamic Island（ノッチ UI）：**
    *   物理的なノッチに沿い、折りたたむとハードウェアになじむパネルです。
    *   ホバーまたはクリックすると、弾むようなスプリングアニメーションと触覚フィードバックを伴って展開します。
    *   回転する目盛りリングと、充電電力に合わせて脈動する発光コアを備えた **JARVIS スタイルのアーク・リアクター HUD**。
    *   電力、バッテリーの健全性、温度を一目で確認でき、クイック切り替え（ウィジェット、リセット、アニメーション、通知）も利用できます。
*   **🌍 ローカライズ：**
    *   英語、トルコ語、簡体字中国語（简体中文）、日本語、韓国語（한국어）の UI を完全にサポートします。macOS のシステム言語に従うか、MacWake の設定で言語を選択してください。変更を適用するには再起動が必要です。WidgetKit 拡張機能には、引き続きシステムが提供する個別の言語環境が適用されます。
*   **📊 詳細なセッション追跡（現在のセッション）：**
    *   画面オン時間とスリープ時間を追跡します。
    *   再起動/シャットダウンを検出し、データの整合性を途切れなく保ちます。
    *   バッテリーが 1% 減るごとの平均画面使用時間を示す効率を計算します。
*   **🖱️ 半透明デスクトップウィジェット：**
    *   ロックしてデスクトップ上の任意の位置に配置できるフローティングウィジェットです。
    *   Apple スタイルの円形バッテリー残量インジケーター。
    *   バッテリー温度とサイクル回数をリアルタイムで監視します。
*   **🔌 スマート電源アダプター分析とハイブリッドアルゴリズム：**
    *   **⚡️ ハイブリッド消費電力：** 電源接続時はシステム全体の消費電力（`SystemPowerIn`）、バッテリー使用時は放電率（`InstantAmperage`）をシームレスに組み合わせ、root（`sudo`）権限なしでリアルタイム（動的）の Watt 消費量をメニューバーに正確に表示します。
    *   接続中のアダプターの公称ワット数（例：30W）と実際の充電状態を監視します。
    *   Apple 純正アダプター検証（MFI Check）。
    *   使用中のポート（MagSafe、USB-C、Thunderbolt）を識別します。
    *   充電効率が低い状況での**低速充電通知**。
    *   過去のすべての充電器の使用回数を追跡するアダプター履歴ログ。
*   **⏰ 急速なバッテリー消費の通知：**
    *   バッテリー使用中、直近 10 分間の急激な残量低下（例：5% 以上）を検出し、すぐにローカル通知を送信します。
*   **💫 iPhone スタイルの充電アニメーション：**
    *   充電ケーブルを接続すると、現在の残量（%）を表示する洗練された全画面トランジションアニメーションが画面中央に現れます（設定で切り替え可能）。
*   **🛡️ スマートバッテリー保護と温度通知：**
    *   バッテリー温度が 38°C のしきい値を超えると、警告カードを即座に表示し、ローカル通知を送信します。
    *   デバイスが 99% 以上の充電状態で 24 時間を超えて連続接続されている場合、バッテリーの健全性を守るため放電警告を表示します。
*   **📈 バッテリー健全性の低下ログ：**
    *   バッテリーの最大容量が変化するたびに、日付とサイクル回数を記録する自動履歴ログです。
    *   過去の容量低下を、ハードウェアタブの洗練されたタイムラインに表示します。
*   **🚀 簡単なアクセスと自動起動：**
    *   ログイン時に自動起動するオプション。
    *   ダーク/ライトモードに対応する高度な動的カラーパレット。

## 🛠️ インストール

### オプション 1 — Homebrew（推奨）

```bash
brew tap Jarvis322/tap
brew install --cask macwake
```

### オプション 2 — 直接ダウンロード

[GitHub Releases](https://github.com/Jarvis322/MacWake/releases) から最新の `Wake-1.0.dmg` をダウンロードし、DMG を開いて **MacWake.app** を Applications フォルダへドラッグしてください。

### 動作要件
*   **macOS 14.0 (Sonoma)** 以降
*   Apple Silicon または Intel

### Terminal の手動コマンド
コマンドラインからアプリを管理する場合：

*   **アプリを起動するには：**
    ```bash
    open /Applications/MacWake.app
    ```
*   **アプリを終了するには：**
    ```bash
    killall MacWake
    ```

## 📂 プロジェクト構成

*   `Sources/MacWakeApp.swift`：アプリケーションのライフサイクル、メニューバー統合、単一インスタンス管理。
*   `Sources/BatteryTracker.swift`：電源状態の追跡（IOKit および IOPS）、セッションデータの保存、通知ロジック。
*   `Sources/MacWakeMenuView.swift`：メイン UI コンポーネントと、メニューバーアイコンのクリック時に表示されるタイムライングラフ。
*   `Sources/WidgetWindow.swift`：フローティングデスクトップウィジェットのウィンドウ、ドラッグ処理、円形インジケーター。
*   `Sources/ChargingAnimation.swift`：充電ケーブルの接続時に起動する全画面アニメーションレイヤー。
*   `Sources/LaunchAgentManager.swift`：macOS の `SMAppService` API を使用したログイン項目の設定。

## 🔒 セキュリティと権限

このアプリケーションは、バッテリー状態と充電アダプターの監視に管理者（root）権限を必要としません。標準の macOS IOKit API のみを使用します。
*   **通知：** 急速な放電の通知を受け取るには、アプリの初回起動時に通知権限を付与することを推奨します（メニューの「有効化/設定」ボタンで管理できます）。
