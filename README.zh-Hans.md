# 🔋 MacWake

[English](README.md) | [简体中文](README.zh-Hans.md) | [Türkçe](README.tr.md) | [日本語](README.ja.md) | [한국어](README.ko.md)

**MacWake** 是一款优雅的 macOS 菜单栏与桌面小组件应用，用于跟踪详细的电池健康状况、使用分析和充电习惯。它使用 Swift 和 SwiftUI 构建，忠实遵循现代 macOS 设计准则（玻璃拟态、鲜明效果）。

## 🖥️ Dynamic Island 与菜单栏

**Dynamic Island** 位于刘海区域：收起时与硬件融为一体，悬停或点击时通过富有弹性的弹簧动效展开，并提供触觉反馈。它会呈现实时功率、电池健康状况和电池温度，同时提供桌面小组件、会话重置、动画和通知的快捷开关。

## ✨ 功能

*   **🔋 充电上限（全系 Apple Silicon M 系列）：**
    *   可将充电上限设置为 50% 至 95% 之间的任意值，减少电池的长期损耗。
    *   根据芯片选择最佳方式：M1/M2/M3 使用简洁的充电抑制（CHTE/CH0C），M4 使用适配器控制（CHIE）——通过一个经过公证的小型后台辅助程序实现（仅需批准一次，无需输入密码）。
    *   **⛵ 航行模式：** 让电量先下降到下限再补充充电，而不是在上限处频繁微充——循环次数更少，发热更低。
    *   **🧪 深度电池校准：** 完成一次完整循环——放电至约 15%、充电至 100%、保持一小时——以重新校准电量计；可按计划执行，也可通过“立即校准”触发（提供实时阶段状态和取消按钮）。
*   **⌨️ 命令行工具：**
    *   从设置中安装 `macwake` CLI，并在 Terminal 中控制充电：`macwake status`、`charging on|off`、`adapter on|off`、`energy auto|low|high`、`fan auto|<rpm>`。
*   **⚡️ 能源模式：**
    *   直接从菜单切换 macOS 能源模式（自动 / 低电量 / 高功率）——仅在支持该模式的 Mac 上显示高功率模式。
*   **🌀 监控标签页——风扇 + 高占用 App：**
    *   风扇状态、一小时转速历史记录和手动目标 RPM（实验性功能——温度超过 92°C 时会自动恢复系统控制）集中在独立标签页中。
    *   按 CPU 或 RAM 对当前运行的 App 排名，并像 Activity Monitor / iStat Menus 一样按需采样。
*   **🎯 智能充电增强功能：**
    *   一键上限预设（60/70/80/90%）、适合出行日的一次性“仅本次充至 100%”覆盖选项，以及按计划充满功能——每天都会在你指定的时间准备好 100% 电量。
    *   温度保护会在超过 40°C 时暂停充电，并在电池冷却后恢复。
    *   可按你选择的电量设置自定义低电量提醒，早于 macOS 自带提醒发出。
*   **⚡️ Shortcuts 与小组件：**
    *   Shortcuts App 操作：设置充电上限、仅本次充至 100%、启动清洁模式、获取电池状态。
    *   原生 macOS 小组件（小号和中号），显示电池环、健康状况、温度和上限状态。
    *   将健康状况/循环历史记录、会话和适配器日志导出为 CSV。
*   **🎛️ 可自定义菜单栏：**
    *   精确选择菜单栏项目显示的内容——图标、电池百分比、功率/时间、预计剩余时间和温度，并可实时预览。
*   **🏝️ Dynamic Island（刘海 UI）：**
    *   面板紧贴物理刘海，收起时与硬件融为一体。
    *   悬停或点击后通过富有弹性的弹簧动画展开，并提供触觉反馈。
    *   **JARVIS 风格的弧形反应堆 HUD**，包含旋转刻度环和随充电功率脉动的发光核心。
    *   一目了然地显示功率、电池健康状况和温度，并提供快捷开关（小组件、重置、动画、通知）。
*   **🌍 本地化：**
    *   完整支持英语、土耳其语、简体中文、日语（日本語）和韩语（한국어）UI。可跟随 macOS 系统语言，也可在 MacWake 设置中选择语言；重启后更改才会生效。WidgetKit 扩展仍使用系统为其单独提供的语言环境。
*   **📊 详细会话跟踪（当前会话）：**
    *   跟踪屏幕点亮时间和睡眠时长。
    *   通过检测重启/关机，无缝保持数据完整性。
    *   计算效率，显示电量每下降 1% 对应的平均屏幕使用时间。
*   **🖱️ 半透明桌面小组件：**
    *   悬浮小组件可锁定，并能放置在桌面任意位置。
    *   Apple 风格的圆形电量指示器。
    *   实时监控电池温度和循环次数。
*   **🔌 智能电源适配器分析与混合算法：**
    *   **⚡️ 混合功耗：** 接通电源时无缝结合系统总功耗（`SystemPowerIn`），使用电池时结合放电速率（`InstantAmperage`），从而在菜单栏中准确显示实时（动态）瓦特功耗，且无需 root（`sudo`）权限。
    *   监控所连接适配器的标称功率（例如 30W）和实际充电状态。
    *   Apple 原装适配器验证（MFI Check）。
    *   识别当前使用的端口（MagSafe、USB-C 或 Thunderbolt）。
    *   针对低效率充电场景的**慢速充电提醒**。
    *   记录适配器历史，以跟踪以往所有充电器的使用次数。
*   **⏰ 电池快速耗电通知：**
    *   使用电池供电时，检测过去 10 分钟内的电量骤降（例如 5% 或更多），并立即发送本地通知。
*   **💫 iPhone 风格的充电动画：**
    *   插入充电线时，屏幕中央会显示优雅的全屏过渡动画并呈现当前电量百分比（可在设置中关闭）。
*   **🛡️ 智能电池保护与温度提醒：**
    *   电池温度超过 38°C 阈值时，立即显示警告卡片并发送本地通知。
    *   如果设备连续插电超过 24 小时且电量保持在 99% 或更高，则发出放电警告以保护电池健康。
*   **📈 电池健康衰减日志：**
    *   每当电池最大容量发生变化时，自动记录日期和循环次数。
    *   在硬件标签页中以精美时间线展示历史容量衰减情况。
*   **🚀 便捷访问与自动启动：**
    *   可选择登录时自动启动。
    *   与深色/浅色模式兼容的高级动态调色板。

## 🛠️ 安装

### 选项 1 — Homebrew（推荐）

```bash
brew tap Jarvis322/tap
brew install --cask macwake
```

### 选项 2 — 直接下载

从 [GitHub Releases](https://github.com/Jarvis322/MacWake/releases) 下载最新的 `Wake-1.0.dmg`，打开 DMG，然后将 **MacWake.app** 拖到“应用程序”文件夹。

### 系统要求
*   **macOS 14.0 (Sonoma)** 或更高版本
*   Apple Silicon 或 Intel

### 手动 Terminal 命令
如果你希望通过终端管理此 App：

*   **启动 App：**
    ```bash
    open /Applications/MacWake.app
    ```
*   **退出 App：**
    ```bash
    killall MacWake
    ```

## 📂 项目结构

*   `Sources/MacWakeApp.swift`：应用生命周期、菜单栏集成和单实例管理。
*   `Sources/BatteryTracker.swift`：电源状态跟踪（IOKit 与 IOPS）、会话数据存储和通知逻辑。
*   `Sources/MacWakeMenuView.swift`：主 UI 组件，以及点击菜单栏图标后显示的时间线图表。
*   `Sources/WidgetWindow.swift`：悬浮桌面小组件窗口、拖动逻辑和圆形指示器。
*   `Sources/ChargingAnimation.swift`：连接充电线时触发的全屏动画层。
*   `Sources/LaunchAgentManager.swift`：使用 macOS `SMAppService` API 配置登录项。

## 🔒 安全与权限

此应用无需任何管理员（root）权限即可监控电池状态和充电适配器；它完全依赖标准 macOS IOKit API。
*   **通知：** 为接收快速耗电提醒，建议在 App 首次启动时授予通知权限（可通过菜单下的“启用/设置”按钮管理）。
